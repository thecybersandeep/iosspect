IOSspect

v0.1.0 iOS 15 / 16.7 LTS / 17 / 26 jailbreak only rootful + rootless arm64

Live runtime audit for installed iOS apps. Runs on the jailbroken device, hosts an HTTPS dashboard, lets you read any app's data container and bundle from your browser.

What it does

  • File browser for each app's Data container and Bundle
  • SQLite reader: tables, schema, paginated rows, ad-hoc SELECT, CSV export
  • NSUserDefaults plist viewer + editor
  • Decoded Info.plist + embedded.mobileprovision
  • Components list: URL schemes, extensions, associated domains, background modes
  • Mach-O scan: per-arch slices, encrypted flag, stripped check, one-click dylib download
  • os_log_stream tail over WebSocket
  • Live process and socket tables
  • One-click pull IPA (base bundle + provisioning)

How to use

  1. Install IOSspect from this repo
  2. Open IOSspect from the home screen
  3. Tap Start server, copy the URL and the browser password
  4. Open the URL in any browser. Verify the SHA-256 fingerprint matches what's on the phone.
  5. Sign in with the password
